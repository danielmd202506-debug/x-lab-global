# X-LAB 北美视觉风格探索报告 / North America Visual Style Exploration

Date: 2026-06-06  
Scope: 北美自行车竞品官网、经销商/B2B 入口、服务与保修页面、用户画像、VOC 信号。  
Output use: X-LAB 官网、Custom Configurator、Dealer route、Store route、B2B Login / Dashboard 的视觉与交互方向输入。

## 1. Executive Summary / 执行摘要

### 中文

北美高性能自行车官网的竞争已经不是单纯的品牌视觉竞争，而是“产品发现 + 授权门店 + 本地服务 + 保修/注册 + B2B 经销商工具”的综合体验竞争。Trek、Specialized、Giant、Cannondale 这类头部品牌用清晰门店网络、服务承诺、在线到店流程和保修路径降低购买风险；Canyon 代表 DTC 效率，但也暴露服务网络焦虑；Marin、PNW、Arundel 等中型或配件品牌的 B2B 入口更直接，强调查库存、下单、账户、经销商支持和质保。

X-LAB 在北美不应只做“暗色高端竞速官网”。更适合的视觉策略是：保留技术、竞速、工厂控制的强品牌气质，同时在页面结构中显性化 dealer-backed delivery、fit review、warranty path、service parts、inventory / RFQ / training portal。换句话说，视觉风格要从“看起来高性能”升级为“看起来高性能，并且有人负责交付和售后”。

补充品牌前提：X-LAB 属于喜德盛 / XDS 的高性能子品牌，因此北美视觉系统不能完全脱离母品牌。X-LAB 应继承 XDS 的核心识别资产，包括红黑高对比色、制造/工程/垂直整合叙事、规模化生产可信度、经销商优先模型和国际赛事背书；但表达方式应更锋利、更高级、更面向高性能骑行用户。换句话说，XDS 是“工程与制造可信母体”，X-LAB 是“性能化、北美化、经销商友好的前台表达”。

### English

The North American bicycle web experience is no longer only a brand showcase. Leading sites combine product discovery, authorized retail, local service, warranty registration, pickup workflows, and B2B dealer operations. X-LAB should keep its technical, race-road, factory-controlled intensity, but make dealer-backed delivery, fit review, warranty path, service parts, RFQ, inventory, training, and launch assets visible in the UI.

## 2A. Parent Brand Inheritance / 喜德盛 XDS 母品牌继承

X-LAB 在北美的设计策略应该先回答一个品牌架构问题：它不是一个孤立新品牌，而是 XDS / 喜德盛长期制造、工程、供应链和赛事能力的高性能前台。

### What X-LAB should inherit / 应继承什么

| XDS mother-brand asset / 母品牌资产 | X-LAB expression / X-LAB 转译方式 |
| --- | --- |
| 红黑高对比识别 | 保留黑、白、红作为主视觉骨架；红色用于主 CTA、速度、风险、赛事线索，不做廉价大面积铺满。 |
| 工程与制造能力 | 用 carbon layup、QC、测试、垂直整合、生产 hub、serial traceability 等证明“不是贴牌故事”。 |
| 规模化供应链可信度 | 在 B2B 页面显性表达 inventory、ETA、allocation、service parts、launch assets。 |
| 经销商优先模型 | 把 dealer-backed delivery、independent shops、fit review、warranty workflow 放到核心模块。 |
| 国际赛事/职业队背书 | 用 WorldTour / XDS Astana / championship track record 作为性能证明，但避免只做赛事情绪。 |
| 中国主品牌身份 | 面向北美时表达为 engineering origin + manufacturing control，不应过度本土化到看不出 XDS 来源。 |

### Visual inheritance rule / 视觉继承规则

- Base colors: XDS red + black + white remain the recognition spine.
- Sub-brand elevation: X-LAB adds carbon gray, technical cyan, operational paper, and restrained status colors.
- Graphic elements: use diagonal speed lines, carbon/factory grids, precision measurement marks, thin rules, serial/inspection labels.
- Avoid: fully replacing XDS red with trendy cycling colors; overusing beige/cream premium palettes; making X-LAB look like an unrelated western boutique brand.
- Tone: XDS gives credibility through scale and manufacturing; X-LAB gives desire through performance, fit, dealer service and riding specificity.

## 3. Competitor Visual Patterns / 竞品视觉风格观察

### 3.1 Trek

Visual mood:

- 大众可信、清晰、零售网络强，视觉不是极端前卫，而是“可购买、可服务、可到店”的成熟体系。
- 高对比产品图片 + 清晰电商模块 + 服务/支持入口。
- 字体多为现代无衬线，层级明确，导航密度高但清楚。

UI signals:

- 强调在线订购、送到家或本地 Trek 零售商组装支持。
- Trek U / B2B 与 Retailer Services、培训、保修流程相关。
- 视觉上偏白底/浅底电商体系，红色或品牌色不作为大面积背景，而作为识别和操作强调。

Implication for X-LAB:

- X-LAB 不需要模仿 Trek 的大众零售温和感，但必须学习它把服务、培训、B2B、质保和门店连接起来的方式。

### 3.2 Specialized

Visual mood:

- Rider-first、强图片、强产品故事，常用大幅骑行/产品图建立情绪。
- 整体更 lifestyle-performance，但仍把 store finder、support、warranty 放在明确位置。

UI signals:

- 门店与 Rider Care 组成购买后信任链。
- 产品视觉更偏“骑手体验 + 专业性能”，不是纯工厂或纯商务。

Implication for X-LAB:

- X-LAB 的公开产品页可以比 B2B 更有速度感和情绪，但每个情绪模块后要接 fit、dealer、warranty、service。

### 3.3 Giant

Visual mood:

- 清晰、电商化、全球大品牌感。图片覆盖 road、mountain、lifestyle、store。
- 视觉更“可靠、可获得、门店友好”，不是极端高压。

UI signals:

- 官方页面强调本地 Giant retailer、服务、fit、骑行体验。
- Store finder 页面有服务能力标签，例如 E-Bike Repair Certified。

Implication for X-LAB:

- X-LAB 的 store/dealer 页面应该展示服务能力标签，而不是只显示门店地址。

### 3.4 Cannondale

Visual mood:

- 强产品图 + 清晰购买路径 + 授权门店。
- 视觉上常用明亮背景和强产品主体，电商路径直接。

UI signals:

- Buy Online Pick Up in Shop 流程明确：线上购买，授权店免费组装，到店取车。
- Dealer locator 支持库存、价格区间、车型过滤等购买决策信号。

Implication for X-LAB:

- X-LAB 的最佳路径不是纯 DTC，而是 Online Config / RFQ -> Dealer Review -> Store Pickup / Service。

### 3.5 Canyon

Visual mood:

- 线上效率、强产品、强技术支持中心。
- 支持页面信息架构清楚：repair、spares、warranty、service locations、serial / part number search、bike specs。

UI signals:

- DTC 的强项是参数透明、支持中心强、备件查找工具化。
- 弱项是用户和车店对服务承接仍有焦虑。

Implication for X-LAB:

- X-LAB 可以吸收 Canyon 的支持中心结构，但要用 dealer-backed service 规避 DTC 售后焦虑。

### 3.6 Marin

Visual mood:

- 更直接、更中型品牌，少一点头部品牌的复杂营销。
- Dealer Portal 页面非常清楚：B2B 用来查库存、下单、管理账户。

UI signals:

- B2B 入口文案短、直接、功能导向。
- 对 X-LAB 的启发非常强：B2B Login 页面不能只是登录框，必须说明登录后能做什么。

### 3.7 Santa Cruz

Visual mood:

- 社区、山地、专业店、真实骑行文化。
- Dealer application 强调经销商应该是市场专家、社区参与者、服务能力强。

UI signals:

- 经销商资格不是“有钱进货”，而是社区影响力、专业服务、组织能力。

Implication for X-LAB:

- Dealer 页面视觉应有“选择合适伙伴”的姿态，而不是单纯招商。

### 3.8 Cervelo

Visual mood:

- 高性能公路/铁三专业品牌，视觉更克制、专业、产品线清晰。
- Store locator、Register Your Bike、Warranty、Product Manuals 等支持入口明显。

UI signals:

- 高端性能用户会接受专业冷静的视觉，但必须能快速找到 dealer、register、warranty、manual。

Implication for X-LAB:

- AD9 / RT9 页面可参考这种专业克制，不必过度 lifestyle。

### 3.9 PNW / Arundel / Brooklyn Bicycle Co. B2B

Visual mood:

- 比整车品牌更工具化，页面目的明确。
- B2B 文案通常直接解释 existing dealers、apply、wholesale pricing、order product、dealer support、manuals、service parts、warranty。

UI signals:

- 中小品牌的 B2B 入口未必视觉复杂，但必须回答“我登录后能干什么”。
- 经销商支持联系方式、审核周期、批发价、订单、质保、资料下载是基本信任信号。

Implication for X-LAB:

- B2B portal 的视觉要更像采购和支持工具，不要像品牌 campaign。

## 4. Color System Trends / 色系趋势

### 4.1 Competitor color behavior / 竞品色彩行为

| Brand Type / 类型 | Common Palette / 常见色系 | Visual Meaning / 视觉含义 |
| --- | --- | --- |
| Trek / Giant / Cannondale | 白底、浅灰、黑字、品牌色局部强调 | 成熟电商、可信购买、门店支持。 |
| Specialized / Santa Cruz | 黑白基础 + 强图片色彩 + 少量品牌红/绿/橙 | Rider culture、速度、户外真实场景。 |

| Marin / PNW / Arundel B2B | 简洁白底、黑字、单色按钮 | 工具化、登录、申请、订货效率。 |

### 4.2 Recommended X-LAB palette / X-LAB 推荐色彩

X-LAB 当前色彩方向是合理的，但需要分场景使用：

- Public brand / product: 深黑 `#050607`、高亮白、X-LAB 红、技术青。
- Dealer/store trust modules: 可以更多使用浅底、白卡、深文字，让 fit、warranty、service、parts 信息更可信。
- B2B dashboard: 浅灰背景 + 白色面板 + 深色导航栏，红色只给高风险/主动作，青色给流程和技术状态。
- 避免把整个北美站做成单一深色；高性能用户喜欢冲击力，但购买/保修/服务信息需要可读的浅色工作区。

Recommended tokens:

| Brand black | `#050607` | Hero、产品、竞速氛围。 |
| Primary ink | `#f7f8f8` / `#080b0d` | 暗底/浅底正文。 |
| Action red | `#d91f2d` | 主 CTA、风险、紧急状态。 |

## 5. Imagery And Illustration Style / 图片与插图风格

### Competitor pattern / 竞品模式

- 头部整车品牌依赖真实产品、真实骑行、真实门店/服务，而不是抽象插画。
- 支持中心和 B2B 页面倾向少图或功能图，重点是信息结构。
- 高性能产品页通常用可识别的整车、部件细节、骑行场景、队车/赛事/测试图。
- 门店与服务页面常用 mechanic、shop、pickup、authorized retailer、support imagery。

### X-LAB recommendation / X-LAB 建议

Use:

- Real product side views.
- Factory layup, QC, testing, logistics, serial traceability.
- Dealer showroom, bike fitting, test ride handoff, service bench.
- Terrain-specific scenes: fast road, climb, all-road/gravel, urban greenway.
- B2B screenshots or structured UI previews for portal modules.

Avoid:

- 手绘插画、抽象渐变背景、无法识别产品的氛围图。
- 过度 lifestyle 的“自由骑行”图库图。
- 只展示高速公路竞速，忽略 gravel / endurance / urban fitness 用户。

Illustration rule:

- 如果没有真实图，用结构化 UI diagram 或 product render，不用松散手绘。
- B2B 页面不需要插画，应该用表格、状态、文件、任务和工作流。

## 6. Typography / 字体与排版趋势

### Competitor pattern / 竞品模式

- 多数北美自行车官网使用现代无衬线，重点是清晰、可购买、可扫描。
- 高性能品牌会使用更强的 condensed / bold heading，但正文仍保持高可读性。
- B2B 和支持页面通常不做大标题戏剧感，而是 15-16px 正文、紧凑表格、清楚标签。

### X-LAB recommendation / X-LAB 建议

Current fonts are suitable:

- Display: `Barlow Condensed`
- Body: `Source Sans 3`

Use them this way:

- Public hero: 短、强、大写、凝练。
- Product / custom: 标题强，但正文必须解释选择逻辑。
- Dealer/store: 标题可以强，但内容更 operational，少 slogan。
- B2B: 标题降低戏剧性，优先扫描效率。

Line length:

- Hero copy: 50-56ch.
- Lead copy: 58-65ch.
- Dashboard cards: 1-2 行内完成意思。

## 9. B2B Portal Style / B2B 门户风格

### Industry pattern / 行业模式

B2B portals in cycling are not brand theaters. They are procurement, inventory, service, and account tools. Marin states the basic dealer portal promise clearly: check stock, place orders, manage account. PNW emphasizes dealer support, service parts, manuals, and warranty. Trek ties B2B to warranty claims and Trek University. QBP represents the deeper distributor model where ordering and inventory efficiency are the core product.

### X-LAB B2B visual direction / X-LAB B2B 视觉方向

Interface qualities:

- Light workspace, dark persistent navigation.
- Compact dashboard cards, not giant hero panels.
- Status-first rows: high, medium, low, released, pending, blocked.
- Tables for orders, RFQ, price, allocation, logistics, warranty, documents.
- Portal lane cards for RFQ, inventory/ETA, price tier, warranty, academy, launch assets.
- CSV/PDF/export/download actions visible.

Avoid:

- Large cinematic hero.
- Marketing copy inside task panels.
- Decorative icons without operational meaning.
- Dashboard cards that only show vanity metrics.

## 10. Industry Trends / 行业趋势

### 10.1 B2B2C is the winning trust model

消费者在线研究，但高端整车仍强依赖本地门店完成 fit、组装、试骑、取车和售后。Cannondale 的 Buy Online Pick Up in Shop、Trek 的本地零售商支持、Giant 的授权零售商服务，都说明“线上效率 + 线下服务”正在成为北美主流。

### 10.2 Warranty and registration are conversion content

保修不再只是 footer support 页面。高端用户会把 warranty、registration、authorized purchase、claim process 当成购买前的信任判断。

### 10.3 Dealer enablement is brand experience

经销商培训、销售素材、服务手册、价格/库存/RFQ 工具，直接影响最终消费者看到的品牌质量。B2B 体验也是品牌体验。

### 10.4 DTC efficiency is expected, but DTC service anxiety remains

Canyon 类 DTC 模型教育了用户期待清晰参数、备件查询和支持中心，但也强化了 service partner、local support、warranty ownership 的重要性。X-LAB 可以吸收 DTC 效率，同时强调 dealer-backed service。

### 10.5 Visual design is moving from “cinematic only” to “proof + workflow”

品牌首页仍需要强视觉，但北美购买决策更看重可验证流程：compare, configure, fit, inventory, pickup, register, claim, service.
